# College Admissions Calculator

**Live site:** https://verialix996.github.io/webcourseEx4/

## What it does

A web form that calculates college admission eligibility based on the following inputs:

- **Age**
- **Psychometric Grade**
- **Bagrut Grade**
- **Mechina Grade**

### Features

- Select between two admission tracks:
  - **Psychometric and Bagrut** – disables the Mechina Grade field
  - **Mechina and Bagrut** – disables the Psychometric Grade field
- All grade and age fields accept numbers only
- Clicking **Validate**:
  - Shows an alert if Bagrut Grade is empty
  - If age is above 30, displays: *"You are eligible for admission to any faculty you choose"* with a green background
